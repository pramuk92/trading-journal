# Plus500 Futures Trading Journal

A Streamlit-based web application for analyzing Plus500 futures trading performance from monthly PDF statements.

## Features

- **PDF Parsing**: Automatically extract trade data from Plus500 monthly statements
- **Performance Analytics**: Win rate, profit factor, average P&L, and more
- **Interactive Visualizations**: P&L charts, win/loss distribution, instrument performance
- **Trade Analysis**: Breakdown by instrument, day of week, and trading session
- **Data Export**: Download analyzed trade data as CSV
